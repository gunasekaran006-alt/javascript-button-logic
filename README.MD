# JavaScript Like Button Logic 📱❤️

A simple, clean, and interactive **Like Button** project created as part of the "Problem of the Day" challenge. This project demonstrates basic DOM manipulation using JavaScript.

## 🚀 Features
- **Toggle Functionality:** Click once to "Like" (Heart turns red) and click again to "Unlike".
- **Real-time Counter:** The like count increases and decreases dynamically.
- **Responsive Design:** A simple and centered UI that looks great on any screen.

## 🛠️ Tech Stack
- **HTML5:** For the structure of the button and card.
- **CSS3:** For styling, hover effects, and animations.
- **JavaScript:** For the logic behind the like/unlike functionality.
